[GO]"..."的用法

2025-02-03

1.在定义数组[…]时的数组容量处的…表示由编译器自动计算数组元素大小

2. 用在append函数的第二个参数后面时，append(a, b…) 表示增加b中所有的数据到a中,append函数原型 func append(s []T, x …T) []T

3. 用在函数形参中时表示可变参数

4. 用在带有可变参数的函数的二次封装时再次调用函数时表示解构参数

  • … 解构参数时，要求传递的必须是切片类型。
